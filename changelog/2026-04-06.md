# 2026-04-06

## 버그 수정

- **Instagram/Threads OG 파싱 429 오류 수정** — Supabase Edge Function(Deno Deploy)은 클라우드 서버 IP를 사용하기 때문에 Instagram이 요청을 차단해 429를 반환하던 문제 해결. Instagram/Threads URL은 OG fetch 및 Gemini 호출을 건너뛰고 `"{platform}콘텐츠"` 고정 문자열을 저장하도록 변경 (`discord-verify/verification.ts`)
- **callGemini 유닛 테스트 stub 오류 수정** — Gemini는 plain text를 반환하는데 테스트 stub이 JSON 형식(`{"summary":"..."}`)으로 잘못 설정되어 테스트가 항상 실패하던 문제 수정 (`tests/discord-verify/services/summarizer.test.ts`)

## 변경사항
